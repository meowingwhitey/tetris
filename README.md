# tetris
C와 Win32 API를 사용하여 구현한 간단한 tetris

* **추후 구현해야 할 것들 및 수정해야 될 버그**
1. 맵의 가장 왼쪽에서 블럭 회전이 안됨
2. 블럭 종류별로 색상 구현
3. 점수에 따라 속도 차등 구현
4. 드롭 기능
5. 블럭 안착을 좀 더 부드럽게
6. 블럭 홀드 기능
7. 다음 블럭 미리보기
